# De Bar

De bar is een web app gebouwd voor mijn vereniging Waterscouting "De IJssel", de app was nodig als een robuust systeem om drankjes af te strepen van ieders saldo (hiervoor werd met excel gewerkt).

## Features
- stripe integratie om saldo op te waarderen
- drankje afstrepen voor je zelf, of voor iedereen
- trakteer pot
- leaderboard
